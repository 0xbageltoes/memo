# Vim

## Search
```shell
/ search
? search-backwards
* search-current-word
# search-current-word-backwards
n repeat-search
N repeat-search-reverse
```

## Motion
```
% move-to-pair
} move-to-next-paragraph
{ move-to-previous-paragraph
H move-to-top-of-screen
L move-to-bottom-of-screen
M move-to-middle-of-screen
```
